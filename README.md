# 🎓 Student Grade Calculator

A Python program that calculates and assigns letter grades to students based on their markss

## 📊 Grading Scale

| Marks Range | Grade |
|-------------|-------|
| 80 – 100    | A+    |
| 70 – 79     | A     |
| 60 – 69     | B     |
| 50 – 59     | C     |
| 30 – 49     | D     |
| 0 – 29      | F     |

## ✨ Features

- Assigns letter grades based on marks
- Lists all passing students (grade ≠ F)
- Highlights top performers (grade = A+)

## 🚀 Usage

```bash
python studentcalculator.py
